# FUTURE_UX_02
A mobile-first UI/UX design project for an appointment booking application created as part of the Future Interns UI/UX Internship Program. This project focuses on improving booking experience, reducing scheduling friction, and creating a clean, user-friendly mobile interface for a local café business.

Mobile App UI for Appointment Booking
Future Interns – UI/UX Internship Task 2
Project Overview

This project is a mobile-first appointment booking application designed for a local café business called The Cozy Corner.

The goal of this project was to create a clean, modern, and user-friendly booking experience that helps customers reserve appointments easily while improving scheduling management for the business.

This project was designed as part of the Future Interns UI/UX Design Internship Program.

Problem Statement

Many local businesses still manage bookings through:

Phone calls
WhatsApp messages
Manual registers

This often causes:

Missed bookings
Scheduling confusion
Poor customer experience
Delayed responses
Difficulty managing reservations

This app solves these problems through a simple and structured mobile booking experience.

Objective

The objective of this project was to:

Design a simple mobile booking experience
Reduce booking friction
Improve usability for first-time users
Create a clean and professional UI
Ensure smooth appointment scheduling on mobile devices
Target Users

The app is designed for:

Café visitors
Students
Working professionals
Customers who prefer advance reservations
First-time mobile users
